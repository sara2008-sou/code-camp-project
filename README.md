<h2>📌 Project Overview</h2>
<p>
This project is an end-to-end data product designed to guide students toward their ideal technical tracks. It features a smart, interactive survey that uses logic to recommend career paths. The user demographic data collected is then analyzed to provide crucial insights for organizations to optimize future scholarship distributions.
</p>

<hr>

<p>
Our analysis phase transforms raw survey data into actionable intelligence. Below is a snapshot of our <b>Power BI</b> analysis dashboard, designed to give a clear view of regional demand and demographics.
</p>

<h2>🚀 Detailed Workflow & Features</h2>
<p>
The project follows a rigorous data lifecycle:
</p>

<ul>
  <li><b>1. User Interaction (The Survey):</b> A beautiful and intuitive web interface built with <b>Streamlit</b>. Users answer questions about their interests and skills, and receive a tailored track recommendation (like "Data Science" or "AI").</li>
  <li><b>2. Data Collection & Management:</b> All survey responses and user demographic data are securely collected and stored in <b>Excel</b>. This serves as the clean data source for all analysis.</li>
  <li><b>3. Insights & Visualization (The Dashboard):</b> Our custom-built <b>Power BI</b> dashboard connects to the Excel data to visualize track trends and regional demand. This dashboard enables strategic planning for future "Code Camp" scholarship opportunities across governorates.</li>
</ul>
